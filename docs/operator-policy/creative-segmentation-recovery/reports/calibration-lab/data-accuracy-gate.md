## What Was Verified

- Creative table and Decision OS use the same current creative source path for the checked rows.
- Exported rows use consistent creative identifiers between table rows and Decision OS rows after sanitization.
- Recent 7-day, mid 30-day, and long 90-day windows were fetched with fixed date windows from the source path, not by treating the selected UI reporting range as action authority.
- Account-relative baselines are present on exported rows and include reliability metadata.
- Campaign-relative baseline summaries in the artifact are lab-computed only and are not treated as production campaign benchmark authority.
- Commercial Truth missing/full state is carried on exported rows through `commercialTruthAvailability`.
- Old-rule challenger output is generated by `buildCreativeOldRuleChallenger` and is not delegated Decision OS output.
- Missing data is emitted as missing evidence or gate blockers instead of being silently treated as a valid judgment dataset.

## Blockers

- `company-01` had snapshot candidates but returned zero current Decision OS rows in the current source path.
- The checked database had no rows in `meta_creative_daily`, so it could not be used as an independent warehouse-level creative fact cross-check.
- Current Creative segmentation still depends on the creative API/snapshot source for this lab. When that path produces zero rows for a sampled company, agent judgment would be calibrating source availability rather than creative policy.

## Warnings

- Some exported rows are useful for source-shape inspection, but the failed company-level row availability check means the dataset is not calibration-authoritative.
- Campaign baselines in the artifact are diagnostic summaries. Production campaign segmentation still requires explicit `benchmarkScope: campaign` input.
- The live source path can trigger preview/snapshot refresh behavior; the helper suppresses local refresh fetches and debug logs so the artifact remains sanitized.

## Decision

Stop calibration at Phase B. Do not run the 10 media-buyer-agent panel until the current Creative source can produce a complete, verifiable sanitized dataset across sampled companies.
