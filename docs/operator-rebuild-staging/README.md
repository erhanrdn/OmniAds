# Operator Rebuild Staging

This directory is a temporary, low-noise reporting area for the operator-surface rebuild.

- `LATEST_REPORT.md` always contains only the current step's latest full report.
- `STATUS.md` tracks the current step, repo state, verification state, and completion state.
- `docs/operator-rebuild/HANDOFF.md` is the durable continuity file and must be kept current as future steps replace the staging report.
- Historical permanent documentation should only be created later if explicitly requested.
